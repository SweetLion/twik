twik

Safe passwords the easy way

Twik for Chrome Twik for Android Twik for Python

Twik is an open source, easy to use password generator and password manager for your desktop and your Android device. Your passwords are always available when you need them, but they are never stored anywhere!

Keeping up with todays password requirements isn’t easy. On the one hand, you want to have different, hard-to-guess passwords for each website and service. On the other hand, it’s hard to remember so many passwords! You can rely on a web service to keep all your passwords, and make them readily available from any device. However, this creates a single point of failure: if this password service is compromised, all your passwords would be leaked!

How does Twik work?

Twik works in a different way. You have to provide a private key that will be stored in your smartphone or computer, and think of a strong master key that will not be stored anywhere. Any time you need to get the password for a website, you must type your master key. The combination of the master key, the private key, and the website will be used to generate a unique, strong password. Since passwords are generated each time, even if one of them is compromised the rest would be safe.

Twik generates the passwords using a chryptographic method (HMAC SHA-1), which always produces the same password for a particular combination of a website, master key and private key, while being not reversible. This is the reason why you can always get your website passwords without actually storing them.

Twik for Google Chrome

Twik extension for Google Chrome allows you to sign up and sign in on your websites using safe passwords in the blink of an eye. Twik will detect the password fields on your websites automatically. When you type your master key, Twik automatically replaces it with the password for that website in particular before submitting the form.

Because some websites require different settings, Twik also allows you to customize the password settings for each website in particular.

Twik for Chrome features include:
  • Several profiles, each with its own private key and settings.
  • Customize password generation for each website (password length and characters).
  • Automatically replaces your master key with the website password in your websites.

Twik for Android

Twik for Android makes it easy to manage your passwords on your smartphone. Twik integrates with any web browser, so that you can generate a password quickly by sharing a website with Twik from the browser.

Twik for Android is 100% compatible with the Twik for Google Chrome extension. You can use the same private and master keys to generate the same passwords than on your desktop browser.

Twik for Android features include:

  • Several profiles, each with its own private key
  • Favicons to easily identify websites
  • Identicons to check that you typed your master key correctly at a glance
  • Share any website from a web browser to generate a password for it
  • Customize password generation for each website (password length and characters)
  • Automatically copy generated passwords to the clipboard

Twik for Python

Twik for Python, developed and maintained by Alexandre Possebom, allows you to generate your passwords blazingly fast from the command line, and it's the perfect tool when you need to generate a password for any application on your desktop. It supports all the features available in the Chrome and Android apps, and it's available in the Python Package Index (pip), so you can install it and start using it in the blink of an eye.
